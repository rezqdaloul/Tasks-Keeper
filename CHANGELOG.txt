VERSION 4.0.0 — MAJOR RELEASE
Status: STABLE — CURRENT
================================================================================

NEW FEATURES
────────────────────────────────────────────────────────────────────────────────

1. SWIPE GESTURES
   - Swipe right on any task row → Mark Complete (green reveal)
   - Swipe left on any task row → Delete (red reveal)
   - Threshold-based: short swipes snap back, full swipes trigger action
   - Touch-cancel safe: interrupted gestures reset cleanly

2. DRAG TO REORDER
   - Long-press the grip handle (⋮⋮) on any incomplete task row
   - Drag up or down to reposition within the list
   - Blue line indicates drop target
   - Reorder saved to undo/redo history

3. GLOBAL SEARCH
   - Search icon on Home screen opens full-screen overlay
   - Searches every user, every topic, every task simultaneously
   - Results show username + topic breadcrumb, due date, time, estimate
   - Tap circle to complete a task directly from results
   - "Go →" button navigates to the task's exact location

4. STREAK + WEEKLY CHART
   - Today screen shows current consecutive-day streak (🔥)
   - 7-day completion bar chart below the streak counter
   - Today's bar highlighted in blue; completed days in green

5. TIME ESTIMATES
   - Optional estimate field per task: 15m / 30m / 1h / 2h / 3h
   - Estimate badge shown on each task row
   - Today screen: total committed time for tasks due today
   - Topic screen: total remaining time across all incomplete tasks

6. DATE/TIME DEFAULTS
   - Add Task sheet pre-fills with today's date
   - Time defaults to next full hour (e.g. 2:58 PM → 4:00 PM)
   - Date and Time fields have clear labels above each input

7. SMART HOME BUTTON
   - Pressing Home from Today/Calendar/Settings restores last topic/user
   - Pressing Home again resets to the main Users page
   - Works correctly when navigating deep into the app before switching tabs

UI IMPROVEMENTS
────────────────────────────────────────────────────────────────────────────────

1. FLOATING PILL TAB BAR (iOS glass style)
   - Pill-shaped capsule with backdrop-filter blur + shadow
   - FAB (+) button sits to the right of the pill, only on task list screen
   - Matches modern iOS app design language

2. iOS-NATIVE DESIGN SYSTEM
   - F2F2F7 system gray background / white card surfaces
   - Separator lines use rgba opacity (not solid colors)
   - All render functions use block-body form (no Babel parse issues)
   - SF Pro font stack

BUG FIXES
────────────────────────────────────────────────────────────────────────────────

- FIXED: Swipe-left delete was silently failing (empty stub instead of prop call)
- FIXED: Subtask input lost focus after every keystroke (SwipeRow was defined
  inside App causing remount on every render — moved to module level)
- FIXED: Drag opacity stuck at 0.4 permanently on mobile Safari (document-level
  dragend + pointerup listeners always clear drag state)
- FIXED: Subtask panel required subtasks to exist before showing add input
  (now always shows the input when panel is expanded)
